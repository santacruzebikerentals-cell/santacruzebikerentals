# Santa Cruz Bike Adventures (v2.3)

Coastal-modern React + Tailwind starter with Square booking links integrated as clickable booking cards.
- West Cliff card includes an image + silent auto-playing video carousel (8s auto-rotate)
- Ready for Netlify (netlify.toml included)

## Quick start (if you have Node)
1. npm install
2. npm run dev
3. Open http://localhost:5173/

## Deploy to Netlify
- Build command: npm run build
- Publish directory: dist

## Replace media
- Replace /public/images/bikepath.jpg with your West Cliff photo
- Replace /public/videos/bikepath.mov with your West Cliff video (same filename)
