# Santa Cruz Bike Adventures (v2)

Coastal-modern React + Tailwind starter with Square booking links integrated as clickable booking cards.

## Quick start (if you have Node)
1. npm install
2. npm run dev
3. Open http://localhost:5173/

## Deploy to Netlify
- Build command: npm run build
- Publish directory: dist

## Notes
- Booking cards link directly to Square booking pages.
- Self-guided tours are implemented as hash routes (#/birdwatching, #/surf-history, #/waterfront)
- Favicon placeholder included at public/favicon.svg
